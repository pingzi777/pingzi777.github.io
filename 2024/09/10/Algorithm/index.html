2413最小偶倍数

Given a positive integer n, return the smallest positive integer that is a multiple of both 2 and n.

本身题目不难，但是重点在于怎么优化，对于奇偶问题可以通过位运算的方式来减少运算时间
对题意进行分析，如果是奇数就翻倍，即二进制左移一位；否则不变。

如果是n是奇数，那么最低有效位为1；如果n是偶数，那么最低有效位是0。1的二进制是0001,通过n & 1位与运算，由此可知，n为奇数时，结果为1；n为偶数时，结果为0。所以通过n << (n & 1)来达到左移0位或1位的效果

2236. Root Equals Sum of Children

You are given the root of a binary tree that consists of exactly 3 nodes: the root, its left child, and its right child.
Return true if the value of the root is equal to the sum of the values of its two children, or false otherwise.

这里对二叉树的定义需要注意一下，__init__里的val是该节点的值，left和right分别对应的是左右节点，之所以不是TreeNode类型的，是因为有可能是叶子节点，而叶子结点的left和right属性应该是None，而不是一个TreeNode对象

1486. XOR Operation in an Array

You are given an integer n and an integer start.
Define an array nums where nums[i] = start + 2 * i (0-indexed) and n == nums.length.
Return the bitwise XOR of all elements of nums.

1、暴力法

range()函数的用法:创建了一个整数列表，range(start, stop[, step])
start: 计数从 start 开始。默认是从 0 开始。例如range（5）等价于range（0， 5）;
stop: 计数到 stop 结束，但不包括 stop。例如：range（0， 5） 是[0, 1, 2, 3, 4]没有5
step：步长，默认为1。例如：range（0， 5） 等价于 range(0, 5, 1)
