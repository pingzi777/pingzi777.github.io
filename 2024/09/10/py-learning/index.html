温度转换

输入格式：the temperature in Celsius
输出格式：return it as an array ans = [kelvin, fahrenheit]
Note：Kelvin = Celsius + 273.15
Fahrenheit = Celsius * 1.80 + 32.00 

class Solution:
    def convertTemperature(self, celsius: float) -> List[float]:
        kelvin = celsius + 273.15
        fahrenheit = celsius * 1.80 + 32.00
        return kelvin, fahrenheit

首先定义了一个solution类,类中定义了方法convertTemperature,这里设置了一个参数celsius。

1、self

self 是 Python 类中的一个特殊关键字，用于指代类的实例。
在类方法中，self作为第一个参数，表示调用该方法的对象本身。
当我们定义一个类的方法时，self是必须的，确保方法能访问对象的属性和其他方法。
Note：当你在类中定义一个实例方法时，Python自动将调用该方法的对象作为第一个参数传递给方法，而这个参数通常被命名为self。你可以使用其他名字来代替self，但 self是一种惯例，用其他名字会使代码不易理解。
例子：

class Person:
    # 初始化方法，使用self访问实例属性
    def __init__(self, name, age):
        self.name = name  # 使用self来创建实例属性 name
        self.age = age    # 使用self来创建实例属性 age

    # 一个简单的方法，返回人的介绍
    def introduce(self):
        return f"Hello, my name is {self.name} and I am {self.age} years old."

    # 修改年龄的方法
    def have_birthday(self):
        self.age += 1  # 使用self来修改实例属性 age
        print(f"Happy Birthday {self.name}! You are now {self.age} years old.")

# 创建一个Person类的实例
person1 = Person("Alice", 30)

# 调用introduce方法
print(person1.introduce())  

# 调用have_birthday方法
person1.have_birthday()
